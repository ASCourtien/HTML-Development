Eclipse Web Development Perspective Tutorial

Creating a new Project

Step 1:
Once Eclipse is open, Select File, New and the Static Web Project
Step 2:
Once open; Enter a name for the project and then click Finish

Creating a new HTML Page

Step 1:
Select File, New and then HTML File
Step 2:
Once the new HTML file is named, select the webcontent folder within the project you are working on and the click Finish

Creating a CSS File

Step 1:
Select File, New and the New CSS File
Step 2:
Enter a file name for the created CSS file and click enter

Changing perspectives

Step 1:
Select "Open Perspective"
Step 2:
Select the deisred option, in this case, Web
